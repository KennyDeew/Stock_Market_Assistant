# StockMarketAssistant — Фронтенд

Платформа для анализа инвестиционных портфелей.

## Технологии
- React + Vite + TypeScript
- Material UI
- Axios
- Recharts

## Установка
1. Убедитесь, что Node.js установлен (версия 18+)
2. Выполните:
   ```bash
   npm install
3. Убедитесь, что бэкенд (.NET) запущен на http://localhost:5200
4. Запустите фронтенд:
   ```bash
   npm run dev

Функции:
Аутентификация
Управление портфелями
Операции с активами
Каталог активов
Графики доходности
