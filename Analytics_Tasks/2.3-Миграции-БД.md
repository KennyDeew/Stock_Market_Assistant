# Задание 2.3: Создание миграций базы данных

**Раздел:** Infrastructure Layer - Database  
**Приоритет:** Высокий  
**Время:** 1 час  
**Зависимости:** 2.1

## Цель

Сгенерировать и применить EF Core миграции.

## Команды

```bash
# Генерация
dotnet ef migrations add InitialAnalyticsSchema -p Infrastructure -s WebApi

# Применение
dotnet ef database update -p Infrastructure -s WebApi
```

## Проверка

```sql
\dt  -- список таблиц
\d asset_transactions
\di  -- индексы
```

## Критерии приёмки

- [ ] Файл миграции в Infrastructure/Persistence/Migrations/
- [ ] Обе таблицы созданы
- [ ] Все индексы созданы
- [ ] Down() откат работает
- [ ] Миграция применена
