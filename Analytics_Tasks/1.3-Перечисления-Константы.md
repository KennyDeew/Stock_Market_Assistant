# Задание 1.3: Реализация перечислений и констант

**Раздел:** Domain Layer  
**Приоритет:** Высокий  
**Время:** 30 минут  
**Зависимости:** Нет

## Цель

Определить доменные перечисления (TransactionType, AnalysisContext) и константы.

## Требования

### TransactionType

```csharp
public enum TransactionType
{
    Buy = 1,   // Покупка
    Sell = 2   // Продажа
}
```

### AnalysisContext

```csharp
public enum AnalysisContext
{
    Global = 1,    // Глобальный анализ (все портфели)
    Portfolio = 2  // Анализ конкретного портфеля
}
```

### DomainConstants

```csharp
public static class DomainConstants
{
    public static class Aggregation
    {
        public const int MaxTopAssetsCount = 100;
        public const int DefaultTopAssetsCount = 10;
        public const int MaxPeriodDays = 365;
    }
    
    public static class Validation
    {
        public const decimal MinQuantity = 0.00000001m; // Поддержка крипты
        public const decimal MinPrice = 0m;
        public const int MaxPortfoliosInComparison = 10;
    }
}
```

## Критерии приёмки

- [ ] TransactionType в Domain/Enums/
- [ ] AnalysisContext в Domain/Enums/
- [ ] DomainConstants в Domain/Constants/
- [ ] Значения enum начинаются с 1 (не 0)
- [ ] Константы организованы по логическим группам
- [ ] XML-комментарии к константам

## Структура файлов

```
Domain/
  Enums/
    TransactionType.cs
    AnalysisContext.cs
  Constants/
    DomainConstants.cs
```
