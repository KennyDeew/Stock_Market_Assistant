# Задание 5.2: DTOs и валидаторы

**Раздел:** Application Layer  
**Приоритет:** Высокий  
**Время:** 2-3 часа  
**Зависимости:** 1.1

## Цель

Определить DTOs и FluentValidation валидаторы.

## Компоненты

### Request DTOs

- GetTopAssetsRequest
- ComparePortfoliosRequest

### Response DTOs

- AssetRatingDto (+ FromEntity)
- TopAssetsResponseDto
- PortfolioHistoryDto
- PortfolioStateDto
- PortfolioComparisonDto
- PeriodDto

### Валидаторы

**GetTopAssetsRequestValidator:**
- startDate < endDate
- endDate <= сегодня
- период <= 365 дней
- top между 1-100
- rankBy = "count" или "amount"

**ComparePortfoliosRequestValidator:**
- portfolioIds не пусто
- макс 10 портфелей
- startDate < endDate

## Критерии приёмки

- [ ] Request DTOs в Application/DTOs/Requests/
- [ ] Response DTOs в Application/DTOs/Responses/
- [ ] Валидаторы в Application/Validators/
- [ ] FluentValidation настроен
