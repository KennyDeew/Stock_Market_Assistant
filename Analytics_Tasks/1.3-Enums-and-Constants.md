# Task 1.3: Implement Enums and Domain Constants

**Section:** Domain Layer  
**Priority:** High  
**Estimated Time:** 30 minutes  
**Dependencies:** None

## Objective

Define domain enums (TransactionType, AnalysisContext) and constants used across the domain.

## Requirements

### TransactionType Enum

```csharp
public enum TransactionType
{
    Buy = 1,
    Sell = 2
}
```

**Usage:** Categorizes portfolio transactions.

### AnalysisContext Enum

```csharp
public enum AnalysisContext
{
    Global = 1,    // Aggregation across all portfolios
    Portfolio = 2  // Aggregation for specific portfolio
}
```

**Usage:** Determines scope of rating aggregation.

### Domain Constants

```csharp
public static class DomainConstants
{
    public static class Aggregation
    {
        public const int MaxTopAssetsCount = 100;
        public const int DefaultTopAssetsCount = 10;
        public const int MaxPeriodDays = 365;
    }
    
    public static class Validation
    {
        public const decimal MinQuantity = 0.00000001m; // Crypto support
        public const decimal MinPrice = 0m;
        public const int MaxPortfoliosInComparison = 10;
    }
}
```

## Acceptance Criteria

- [x] `TransactionType` enum exists in `Domain/Enums/`
- [x] `AnalysisContext` enum exists in `Domain/Enums/`
- [x] `DomainConstants` class exists in `Domain/Constants/`
- [x] Enum values start from 1 (not 0) for explicit intent
- [x] Constants are organized by logical groups
- [x] All constants have XML documentation comments
